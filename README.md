# DevOps — вопросы для собеседований

Сборник вопросов для подготовки к техническим собеседованиям на позицию DevOps.

## Оглавление

- [[Git]] — Git и система контроля версий
- [[Linux]] — Linux, процессы, память, bash
- [[Networks]] — Сети, TCP/UDP, DNS, HTTP, SSH
- [[Containers]] — Docker и контейнеры
- [[IaC]] — Ansible и Terraform
- [[CI-CD]] — Continuous Integration / Delivery / Deployment
- [[Monitoring]] — Логи, метрики, Prometheus, ELK, SLO/SLA/SLI
- [[Hardware]] — Железо, IPMI, KVM, сенсоры
- [[Debug]] — 502, chmod, df/du, strace, диагностика
- [[Databases]] — MySQL, PostgreSQL, индексы, репликация
- [[Python]] — Хеш-таблицы, итераторы, декораторы
- [[Kubernetes]] — Pod, Deployment, Service, Helm
- [[AWS]] — EC2, VPC, S3, EKS
