<details>
<summary>Что такое хеш-таблица? Примеры использования.</summary>

Структура данных, где индекс/адрес элемента вычисляется хеш-функцией. Позволяет O(1) доступ по ключу. В Python — `dict`. Пример: маппинг чит-кодов в играх, кэши, счётчики.
</details>

<details>
<summary>Что такое итератор и генератор?</summary>

- **Итератор** — объект с `__iter__` и `__next__`, перебирает элементы коллекции.
- **Генератор** — итератор, создаётся функцией с `yield`. Значения создаются по мере запроса, не хранятся все сразу.
</details>

<details>
<summary>Разница между кортежем и списком?</summary>

- Список — изменяемый (`append`, `remove`), кортеж — неизменяемый.
- Кортеж быстрее, меньше памяти.
- Кортеж хешируем, может быть ключом в dict.
- Синтаксис: `[]` vs `()`; кортеж из одного элемента: `(1,)`
</details>

<details>
<summary>Что такое декоратор?</summary>

Функция, которая принимает другую функцию и возвращает «обёртку» с дополнительным поведением. Пример: логирование, кэширование, проверка прав. Синтаксис `@decorator`.
</details>

<details>
<summary>Какое значение у переменной без значения в Python?</summary>

`None`
</details>
